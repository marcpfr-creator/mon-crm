Héberger sur Netlify pour accès iPhone
Une fois configuré, hébergez ce fichier HTML sur Netlify pour y accéder depuis votre iPhone.
1. Téléchargez ce fichier HTML sur votre ordinateur
2. Allez sur app.netlify.com/drop
3. Créez un dossier, mettez-y le fichier HTML, renommez-le index.html
4. Glissez le dossier dans Netlify Drop → URL générée automatiquement
5. Sur iPhone Safari : ouvrez l'URL → Partager → "Sur l'écran d'accueil"
6. L'app s'installe comme une vraie appli 📱
